# CSAW'25 Redesign

A next-generation redesign of [csaw.io](https://www.csaw.io/) — dark, futuristic, cybersecurity-themed.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. **Framework Preset**: Other
4. **Output Directory**: leave blank
5. Click Deploy ✓

## Local Preview

Just open `index.html` in your browser — no build step needed.

## Stack

- Pure HTML / CSS / JS — zero dependencies
- IBM Plex Mono + Space Grotesk + Bebas Neue (Google Fonts)
- Canvas-based binary rain animation
- CSS animations: glitch, scanline sweep, particle field, hex pulse
